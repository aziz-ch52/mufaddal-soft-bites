# 🍞 Mufaddal Soft Bites | Premium Artisan Bakery

A **high-conversion, responsive landing page** built for an artisan bakery based in **Dahod, Gujarat**.  
This project acts as a modern digital storefront focused on **local engagement**, **trust-building**, and **WhatsApp-based ordering**.

The goal is simple: turn visitors into customers with clarity, speed, and confidence.

---

## ✨ Key Features

### 🎯 Conversion-Focused Design
- Dedicated **Bestsellers** section to reduce decision fatigue
- **Price Anchor** strategy to guide customer choices
- Clear, mobile-first call-to-action buttons

### 🛡️ Trust-Building Elements
- Trust badges: *Hygienic Kitchen*, *FSSAI Certified*
- Verified local customer reviews
- Simple **3-step “How to Order”** process

### 🎨 Modern UI / UX
- Sticky navigation header
- Smooth scroll-reveal animations
- Fully responsive grid layout (mobile & desktop)

### 📍 Local SEO Optimization
- Location-based keyword targeting
- Area-specific customer reviews
- SEO-friendly structure for local discoverability

### ⚙️ Interactive Components
- Functional mobile hamburger menu
- Embedded Google Maps for store navigation
- WhatsApp integration for instant ordering

---

## 🛠️ Tech Stack

- **HTML5**  
  Semantic structure for accessibility and SEO

- **CSS3**  
  Custom styling using CSS Variables (`:root`), Flexbox, Grid, and transitions

- **JavaScript (Vanilla)**  
  Scroll-reveal animations and mobile menu toggle logic

---

## 📌 Purpose

This project demonstrates how **modern UI/UX**, **conversion psychology**, and **local SEO** can be combined to create a practical, business-driven landing page for small local brands.
